title: Emberjs动态加载Bootstrap
tag:
    - Emberjs
    - npm
    - javascript
---

安装命令：
```shell
ember install ember-cli-sass  
ember install ember-cli-bootstrap-sassy  
```
然后修改Ember项目的`style/app.css`为`style/app.scss`，并引入bootstrap。
```scss
@import "bootstrap";  /* 导入bootstrap */
```
在第一行导入bootstrap。
